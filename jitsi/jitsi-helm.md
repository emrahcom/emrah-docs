# Jitsi Helm Chart

Tested on `Debian 13 Trixie` with `minikube`.

### Cluster

Start `minikube`:

```bash
minukube start
```

### Helm Chart Repository

Add `jitsi-helm` repo:

```bash
helm repo list
helm repo add jitsi https://jitsi-contrib.github.io/jitsi-helm/
```
