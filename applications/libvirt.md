## libvirt & virt-manager

Tested on `Debian Bullseye`.

#### installation

```bash
apt-get install virt-manager --install-recommends
```

#### configuration

```bash
adduser my-account libvirt
```

#### virt-manager

`ctrl_l + alt_l` are the grap keys.

```bash
virt-manager
```
