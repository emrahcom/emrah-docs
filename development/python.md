# Python

## Packages

As `root`:

```bash
apt-get install python3 python3-pip python3-venv
```

## Environment

To create:

```bash
python3 -m venv ~/.config/python3/base
```

To load:

```bash
source ~/.config/python3/base/bin/activate
```

To run:

```bash
which python3
which pip3

pip3 list
```
